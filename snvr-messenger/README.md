# 스너버메신저 (웹)

텔레그램 봇과 **같은 백엔드**를 쓰는 웹 클라이언트. 텔레그램 차단 시 2초 전환용.

## 사용 방법

1. **백엔드 실행**  
   `backend` 폴더에서 `npm start` (기본 http://localhost:3000)

2. **텔레그램 봇에서 연동 코드 받기**  
   봇에게 `/link` 입력 → 6자리 코드 수신 (5분 유효)

3. **웹에서 로그인**  
   - `index.html` 을 브라우저로 열거나, 정적 서버로 제공  
   - 백엔드 주소 입력 (기본 http://localhost:3000)  
   - 봇에서 받은 6자리 코드 입력 후 로그인

4. **같은 계정**으로 잔액·송금·받기·스왑·믹싱 사용

## 웹으로 실행하는 방법 (필수)

파일을 그냥 열면(`file://`) 백엔드 호출이 막혀서 동작하지 않습니다. **아래 중 하나**로 웹 서버를 띄운 뒤 브라우저에서 접속하세요.

### 방법 1 — 스너버메신저 폴더에서 (추천)

1. 터미널(또는 VS Code 터미널)에서 **스너버메신저 폴더로 이동**:
   ```
   cd snvr-messenger
   ```
2. 서버 실행:
   ```
   npm start
   ```
3. 터미널에 나오는 주소로 접속 (예: **http://localhost:5000**).  
   브라우저 주소창에 `http://localhost:5000` 입력하면 스너버메신저 화면이 뜹니다.

### 방법 2 — 프로젝트 루트에서

1. 프로젝트 루트(aztec-dev)에서:
   ```
   npx serve snvr-messenger -l 5000
   ```
2. 브라우저에서 **http://localhost:5000** 접속.

### 방법 3 — Python 있는 경우

1. `snvr-messenger` 폴더에서:
   ```
   python -m http.server 5000
   ```
2. 브라우저에서 **http://localhost:5000** 접속.

---

**정리**: `snvr-messenger` 폴더로 가서 **`npm start`** 한 번 실행 → 브라우저에서 **http://localhost:5000** 열면 됩니다.

## 쿼리 파라미터

- `?api=http://localhost:3000` — 백엔드 주소 지정 (페이지 로드 시 기본값)
